# 跳一跳自动化助手 - 打包说明

## 一、安装打包工具

```bash
pip install pyinstaller
```

## 二、打包成 exe

### 基础打包命令
```bash
pyinstaller --onefile --windowed --name=跳一跳助手 main.py
```

### 推荐打包命令（包含图标）
```bash
pyinstaller --onefile --windowed --name=跳一跳助手 --icon=icon.ico main.py
```

## 三、打包参数说明

| 参数 | 说明 |
|------|------|
| `--onefile` | 打包成单个 exe 文件 |
| `--windowed` | 运行时不显示控制台窗口 |
| `--name=跳一跳助手` | 设置 exe 文件名 |

## 四、使用说明

### 使用前准备
1. 确保电脑已安装微信或微信开发者工具
2. 打开微信小程序"跳一跳"
3. 将游戏画面最大化显示

### 软件操作
1. 运行 `跳一跳助手.exe`
2. 设置屏幕分辨率（根据实际显示器）
3. 点击"取色"按钮，将鼠标移到棋子上，按空格键获取棋子颜色
4. 调整按压时间系数（建议先从 1.8 开始测试）
5. 设置跳跃间隔（建议 0.5 秒）
6. 点击"开始"按钮开始自动化
7. 点击"停止"按钮停止自动化

### 参数调优
- **按压时间系数**：跳得太近增大，跳得太远减小
- **跳跃间隔**：设置太小可能导致识别失败
- **屏幕分辨率**：设置错误会导致定位不准确

## 五、分发方式

### Windows 用户分发
1. 打开 `dist` 目录
2. 找到生成的 `跳一跳助手.exe`
3. 使用 WinRAR 或 7-Zip 压缩成 zip 文件
4. 通过微信、QQ 等工具发送给其他用户
5. 对方解压后双击运行即可（无需安装 Python）

### 注意事项
- exe 文件可能被杀毒软件误报，这是正常现象
- 首次运行可能需要管理员权限
- 部分杀毒软件需要添加信任

## 六、常见问题

### 打包失败
- 确保 Python 版本为 3.8 或更高
- 使用 `pip install --upgrade pip` 更新 pip
- 删除 `build` 和 `dist` 目录后重新打包

### 运行错误
- 检查是否正确取色
- 尝试调整按压时间系数
- 确保游戏画面完全可见

### 分辨率设置
- 1920x1080：常见 1080P 显示器
- 2560x1440：2K 显示器
- 3840x2160：4K 显示器
- 其他：根据实际显示器分辨率设置
