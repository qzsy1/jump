# 跳一跳自动化助手

一个基于 Python 的跳一跳游戏自动化工具，帮助您轻松获得高分。

## 功能特点

- 自动识别棋子和平台位置
- 可调节按压时间系数，适应不同屏幕
- 支持屏幕分辨率设置
- 智能取色功能，自动识别棋子颜色
- 简洁美观的界面设计

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

### 1. 打开游戏
在 Windows 端打开微信小程序"跳一跳"，确保游戏画面完整显示。

### 2. 运行程序
```bash
python main.py
```

### 3. 配置参数

| 参数 | 说明 | 推荐值 |
|------|------|--------|
| 屏幕分辨率 | 显示器的分辨率 | 1920x1080 / 2560x1440 |
| 按压时间系数 | 跳跃距离的计算系数 | 1.8 |
| 跳跃间隔 | 每次跳跃后的等待时间(秒) | 0.5 |
| 棋子颜色 | 棋子的RGB颜色值 | 点击"取色"获取 |

### 4. 取色操作
1. 点击"取色"按钮
2. 将鼠标移到游戏中的棋子上
3. 按空格键获取颜色
4. 颜色值会自动填入输入框

### 5. 开始自动化
点击"开始"按钮，程序将自动进行跳跃。点击"停止"按钮可随时停止。

## 参数调优建议

### 跳得太远/太近
- 跳得太远：减小按压时间系数
- 跳得太近：增大按压时间系数
- 每次调整幅度 0.1 左右

### 识别失败
- 检查屏幕分辨率设置是否正确
- 重新取色
- 增大跳跃间隔时间

## 注意事项

1. 首次使用请先测试几次，找到合适的参数
2. 确保游戏画面不被其他窗口遮挡
3. 建议在游戏开始几跳后再启动自动化
4. 不要长时间连续使用，避免账号风险
